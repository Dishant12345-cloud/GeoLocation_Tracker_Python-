#  Geolocation Tracker

This is a simple Python-based Geolocation Tracker project developed during my internship.  
It fetches the user's location using an IP-based API and displays it on an interactive map.

##  Features
- Detects user location using IP address
- Displays City and Country
- Shows location on a map using Folium
- Easy to use and beginner-friendly

##  Technologies Used
- Python
- Requests Library
- Folium Library

##  How to Run the Project

1. Install required libraries:
   pip install requests folium

2. Run the Python file:
   python main.py

3. Open the generated file:
   location.html

##  Output
- Shows location details in terminal
- Opens a map with a marker at your location
